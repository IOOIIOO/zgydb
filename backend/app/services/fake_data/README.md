# Fake Data 目录说明

## 用途
此目录存放所有 AI 模型调用的假数据占位函数。在开发阶段，所有模型接口调用这些假数据函数，确保前后端可在无真实模型的情况下联调。

## 三条铁律
1. **所有返回假数据的函数，函数名必须以 `FAKE_` 开头**
2. **所有假数据函数统一放在此目录下**
3. **每个假数据函数必须包含注释**：功能描述、TODO（替换为真实模型）、预期模型类型、调用位置

## 删除时机
正式部署时，**删除整个 fake_data 目录**，将 `app/services/model_interface.py` 中的导入从 fake_data 切换为真实模型。

## 文件说明

| 文件 | 对应模型 | 功能 |
|------|----------|------|
| `multimodal.py` | M1 - 多模态大模型 | 从 PDF/图片提取结构化信息 |
| `general_llm.py` | M2 - 通用大模型 | 对话提取、能力评分、标签推断、推荐理由撰写、报告润色 |
| `classifier.py` | M4 - 分类模型 | 证书分级 + 竞赛分级 |
| `embedding.py` | M5 - Embedding 模型 | 文本向量化 + 余弦相似度排序 |
| `search.py` | M3 - 通用大模型+联网搜索 | 行业趋势搜索 + 学习资源搜索 |
