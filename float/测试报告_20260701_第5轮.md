## 三、新发现问题

### 🔴 P0 — 阻断级

#### 问题1：推荐理由全部为空

- **现象**：10个岗位的 `recommendation_reason` 都是空字符串
- **影响**：用户看不到为什么推荐这个岗位，无法做决策
- **测试数据**：
  ```
  pos0: id=3026 title=前端开发 score=68.2 reason=
  pos1: id=1118 title=前端开发 score=67.7 reason=
  pos2: id=851 title=前端开发 score=67.1 reason=
  ```
- **疑似根因**：`batch_write_recommendations` 返回空字符串或异常被捕获后返回空

#### 问题2：岗位名称重复

- **现象**：10个岗位全是"前端开发"，只是公司不同
- **影响**：没有去重，用户体验差
- **测试数据**：
  ```
  同名重复: {'前端开发': 10}
  ```
- **建议**：按岗位名称去重，每个岗位名称最多出现3次

#### 问题3：趋势来源URL为空

- **现象**：6个 `info_sources` 的 `url` 字段都是空的
- **影响**：只有标题，没有链接，用户无法查看原始资料
- **测试数据**：
  ```
  - 做计算机前端有前途吗 | url=
  - 纯切图岗已死,2026年前端真正值钱的能力是什么 | url=
  - 前端工程师就业环境参考,详细版!!!就业必看 | url=
  ```
- **疑似根因**：LLM 返回的 JSON 中 url 字段为空，或解析失败

#### 问题4：发展路径资源URL部分为空

- **现象**：6个资源中，部分 `url` 为空
- **影响**：用户无法访问学习资源
- **测试数据**：
  ```
  - 《JavaScript高级程序设计》 | url=
  - 阮一峰 HTML 入门教程 | url= https://wangdoc.com/html/
  - freeCodeCamp | url=
  ```
- **疑似根因**：LLM 返回的 JSON 中部分 url 字段为空

### 🟡 P1 — 严重

#### 问题5：match_score 区分度低

- **现象**：10个岗位的分数范围 63.70~68.20，差值仅 4.50
- **影响**：无法有效区分岗位匹配程度
- **测试数据**：
  ```
  match_score: min=63.70 max=68.20 差=4.50
  ```
- **疑似根因**：embedding 余弦相似度集中在 0.5~0.7，映射到 0~100 后区分度低
