# 岗位详情面板重排

## 目标
DirectionSelect 右侧面板拆分为三个独立卡片：基本信息、匹配分析、岗位详情。

## 改动范围
仅 `frontend/src/pages/dashboard/DirectionSelect.tsx`，不改后端。

## 布局

### 卡片 1: 岗位基本信息
- 标题 + 综合匹配度百分比（右上角）
- 标签行：📍城市 🏭行业 💰薪资 🎓学历

### 卡片 2: 匹配分析
- 三维度匹配（知识/工具/项目），每项带进度条对比 + verdict 标签
- 学历匹配行
- 💡 推荐理由

### 卡片 3: 岗位详情
- description 原文，保留换行
- 优势（绿色卡片）+ 建议提升（黄色卡片），两列并排

### 底部
- "选定此岗位，进入深度规划" 按钮保持不变

## 约束
